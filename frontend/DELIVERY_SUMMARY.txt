GETTING STARTED:
================================================================================

1. Install dependencies (each app):
   cd admin-app && npm install
   cd rider-app && npm install
   cd driver-app && npm install

2. Create .env files from .env.example templates

3. Start development servers (separate terminals):
   cd admin-app && npm run dev   (http://localhost:3000)
   cd rider-app && npm run dev   (http://localhost:3001)
   cd driver-app && npm run dev  (http://localhost:3002)

4. Backend API must run at http://localhost:8000
