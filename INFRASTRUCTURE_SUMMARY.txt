HOW TO GET STARTED
────────────────────────────────────────────────────────────────────────────

LOCAL DEVELOPMENT (5 minutes):
  1. cd infrastructure/docker
  2. docker-compose up -d
  3. Open http://localhost:8000 (API Gateway)
  4. Open http://localhost:3000 (Grafana)

KUBERNETES DEPLOYMENT (15 minutes):
  1. Update kubernetes/secrets.yaml with credentials
  2. kubectl apply -f infrastructure/kubernetes/namespaces/
  3. kubectl apply -f infrastructure/kubernetes/secrets.yaml
  4. kubectl apply -f infrastructure/kubernetes/storage/
  5. kubectl apply -f infrastructure/kubernetes/data/
  6. kubectl apply -f infrastructure/kubernetes/services/

GITHUB ACTIONS SETUP (10 minutes):
  1. Add secrets to GitHub repository
  2. Push to develop → auto-deploys to dev cluster
  3. Push to main → builds Docker images
  4. Manual trigger for staging/production
